# Caffeinated Java ☕

A collection of Java programs and concepts I am practising while preparing for QA Automation interviews.

## Topics Covered

* Java Basics
* Conditional Statements
* Loops
* Arrays
* Strings
* OOP Concepts
* Collections Framework
* Exception Handling
* File Handling
* Java 8 Features

## Prerequisites

* JDK 17+
* Eclipse IDE or VS Code

## How to Run

### Using Eclipse

1. Import the project into Eclipse.
2. Navigate to the desired Java class.
3. Right-click the file.
4. Select **Run As → Java Application**.

### Using Command Line

```bash
javac FileName.java
java FileName
```
